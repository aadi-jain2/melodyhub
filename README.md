# MelodyHub Website

A professional student-led singing initiative website promoting music education and bringing the joy of singing to underserved communities.

## About MelodyHub

MelodyHub is a student-led initiative dedicated to promoting music education and bringing the joy of singing to underserved communities, one voice at a time.

## Website Structure

- **Home** (`index.html`) - Hero section, mission, and program preview
- **Our Story** (`about.html`) - Organization history and timeline
- **What We Do** (`programs.html`) - Detailed program descriptions
- **Impact** (`impact.html`) - Testimonials and transformation stories
- **Gallery** (`gallery.html`) - Photos and videos from performances and events
- **Founder** (`founder.html`) - Founder biography and values

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Responsive Design

## Setup

Simply open `index.html` in a web browser to view the website locally.
